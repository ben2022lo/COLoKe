# COLoKe: Conformal Online Learning of Deep Koopman Linear Embeddings

This repository contains the official implementation of **COLoKe**, a framework for *conformal online learning* of Koopman-invariant representations of nonlinear dynamical systems from real-time and streaming data. COLoKe leverages **statistical conformal principles** to decide *when* to update the model, enabling selective, efficient, and overfitting-free online updates.


## Key elements
- **Online Koopman learning** with adaptive updates  
- **Conformal-based thresholding** for triggering model updates only when necessary  
- **Sublinear Dynamic regret bound** providing theoretical guarantees
